# Команды и сценарии

## Навигация

По умолчанию используйте кнопки в главном меню:

- `🎨 Генерация` — запуск новой генерации, повтор, пресеты.
- `📦 Модели` — скачивание, обновление списка, удаление локальных моделей.
- `⚙️ Сервис` — очередь, настройки, обучение, активные задачи.
- `🛑 Отмена` — отмена активной операции.

Команды остаются доступными как быстрые алиасы.

## Список команд

| Команда | Что делает |
|---|---|
| `/start` | Показывает главное меню и список возможностей. |
| `/help` | То же, что `/start`. |
| `/generate` | Создаёт новую сессию генерации и открывает редактор параметров. |
| `/repeat` | Загружает параметры последней генерации (seed случайный) и открывает редактор; работает и после перезапуска бота. |
| `/presets` | Открывает библиотеку пресетов пользователя. |
| `/download` | Пошаговый поиск и скачивание моделей из CivitAI/HuggingFace. |
| `/models` | Запрашивает ComfyUI `/object_info` и обновляет внутренний кэш моделей. |
| `/queue` | Показывает состояние очереди ComfyUI (`running/pending`). |
| `/jobs` | Показывает активные задачи пользователя (генерация/улучшение). |
| `/settings` | Открывает интерактивные настройки (дефолты генерации и дефолты фильтров поиска моделей). |
| `/training` (`/learn`) | Открывает многостраничное обучение с режимами `Простой/Расширенный`. |
| `/cancel` | Отменяет активное действие: генерацию, скачивание, ввод в FSM. |

## Сценарий 1: базовая генерация

1. Вызовите `/generate`.
2. Выберите checkpoint.
3. Введите Positive/Negative.
4. При необходимости настройте размер, steps, CFG и другие параметры.
5. Нажмите «Генерировать».
6. Дождитесь превью и (при необходимости) запросите PNG без сжатия.

## Сценарий 2: работа с LoRA

1. В редакторе откройте LoRA-меню.
2. Добавьте LoRA и выберите strength.
3. Бот покажет предупреждение о возможной несовместимости с checkpoint (по base model).
4. Если для LoRA есть trained words в метаданных, бот предложит добавить их в Positive.

## Сценарий 3: референсы

1. В редакторе откройте меню референсов.
2. Отправьте фото (или image-документ) в чат — оно добавится в список.
3. Во время генерации бот:
   - использует IP-Adapter, если узлы поддерживаются сервером;
   - иначе делает fallback в img2img.
4. Можно менять силу влияния референса (`reference_strength`).

## Сценарий 4: Smart Prompt

1. Убедитесь, что установлены зависимости `tipo-kgen`, `transformers`, `torch`.
2. В `.env` включите `SMART_PROMPT_PROVIDER=tipo`.
3. В редакторе нажмите «Smart».
4. Отправьте описание (можно с картинкой/картинками).
5. Получите теговые Positive/Negative, сгенерированные TIPO.
6. Выберите действие: применить, объединить, отредактировать или отменить.

## Сценарий 5: скачивание моделей (`/download`)

Диалог состоит из шагов:

1. Тип модели (checkpoint, LoRA, upscaler, embedding, controlnet, vae).
2. Источник (CivitAI, HuggingFace, оба).
3. Фильтры (сортировка, период, base model, NSFW, автор(ы), профили поиска, размер страницы).
4. Текст запроса или прямая ссылка на модель.
5. Постраничный просмотр результатов: навигация, подгрузка, возврат к результатам из подтверждения.
6. Выбор результата и (для CivitAI) версии.
7. Подтверждение и скачивание с прогрессом.

После успешной загрузки бот вызывает refresh списка моделей ComfyUI.

## Сценарий 6: управление задачами (`/jobs`)

1. Откройте `/jobs` или `Сервис -> Мои задачи`.
2. Нажмите на задачу для детальной карточки (`обновить`, `к источнику`, `отменить задачу`).
3. Кнопка `❌ Отменить все` в списке задач отменяет только активные задачи пользователя
   (генерация/улучшение) и не завершает редактор/другие FSM-сессии.

## Сценарий 7: копирование и вставка промпта

1. В редакторе откройте «📋 Копировать/вставить».
2. Нажмите «📤 Копировать» — бот сформирует код обмена с полным набором параметров.
3. Передайте код другому пользователю (или сохраните себе).
4. Для загрузки нажмите «📥 Вставить» и отправьте код текстом или `.txt`-файлом.
5. Параметры редактора будут полностью заменены данными из кода обмена.

## Особенности интерфейса

- Есть два режима: «Простой» и «Про».
- В про-режиме доступны тематические подменю и больше быстрых параметров.
- Для длинных списков используется пагинация.
- Большинство операций можно отменить через `/cancel` или кнопки отмены в интерфейсе.

## Сценарий 8: текстовое обучение

1. Выполните `/training` (или `/learn`).
2. Переключайте страницы кнопками и выбирайте режим `Простой`/`Расширенный`.
3. Прочитайте короткий план: что нажимать в каком порядке.
4. Сразу повторите шаги из блока «Быстрый старт» через `/models` -> `/generate`.
5. Если в процессе увидите ошибку, сверяйтесь с `docs/TROUBLESHOOTING.md`.

## Частые сообщения бота и что делать

- `Нет предыдущей генерации. Используйте /generate.` -> Сначала создайте хотя бы одну генерацию.
- `На сервере ComfyUI нет ни одного checkpoint.` -> Добавьте checkpoint и обновите `/models`.
- `Я не ожидаю текст. Откройте меню кнопками ниже.` -> Бот сейчас не в режиме ввода, выберите действие в главном меню.
- `Сессия истекла. Запустите /download заново.` -> Повторно вызовите `/download` и пройдите шаги поиска.
