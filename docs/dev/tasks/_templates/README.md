# Task Artifact Templates

This directory stores reusable templates for task workflow artifacts.

## Canonical templates

- Plan template: `docs/dev/tasks/_templates/task_plan_template.md`
- Stage report template: `docs/dev/tasks/_templates/stage_completion_report_template.md`

## Usage contract

- These files are source templates and are never used as live task artifacts.
- Live task plans are created under `docs/dev/tasks/{short_task_name}`.
- Live stage reports are created under `docs/dev/tasks/{short_task_name}/reports`.
- The folder name `_templates` is reserved for templates.
