## Шаги ручной проверки

1. Выполните `python -m pytest -q tests/test_localization_resources.py` и убедитесь, что тесты паритета ключей и dynamic discovery проходят.
2. Откройте `docs/LOCALIZATION.md` и проверьте, что инструкция добавления языка описывает file-only сценарий (`metadata + messages`) без правки кода ядра.
3. Проверьте `README.md`, что в разделе документации есть ссылка на `docs/LOCALIZATION.md`.
